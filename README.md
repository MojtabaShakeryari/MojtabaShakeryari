# Featured Research Project 🌍

# Lake Urmia Environmental Monitoring (2016–2025)

An integrated satellite remote sensing and machine learning framework for assessing Lake Urmia water dynamics and climate drivers.

This project investigates long-term changes in lake surface water extent and evaluates relationships between water dynamics and climatic variables using Earth Observation data and environmental modelling approaches.

## Objectives

- Monitoring Lake Urmia surface water changes (2016–2025)
- Assessing climate drivers influencing lake dynamics
- Integrating satellite and climate datasets
- Developing predictive machine learning models

## Data Sources

🛰️ Sentinel-2 MSI satellite imagery  
🌎 Google Earth Engine workflow  
🌧️ CHIRPS precipitation dataset  
🌡️ ERA5-Land temperature data  
💧 MOD16 evapotranspiration dataset  
📉 SPEI drought indicators  

## Methods

- MNDWI and NDVI water extraction
- Satellite image preprocessing
- Time-series analysis
- Climate-water relationship assessment
- Random Forest modelling
- Regression analysis
- Future forecasting

## Main Outputs

- Lake surface water change analysis
- Climate variability assessment
- Machine learning model evaluation
- Environmental prediction framework
- Publication-style scientific figures
